## Alcance cubierto

La documentación contempla el alcance completo del proyecto en tres fases:

- Fase 1: MVP de marketplace.
- Fase 2: mejora de experiencia, confianza y soporte.
- Fase 3: comunidad, publicaciones, seguidores y diferenciación.

Estos documentos sirven como base para desarrollo, revisión con cliente, planificación de milestones y futura implementación.
